NODE EOMP API

HTTP Methods Path Description
GET / Fetches landing/home page
HTTP Method Path Description
GET /users Fetches all Users data
GET /users/:id Fetches corresponding user's data to id specified
POST /users/register Adds a new user
PATCH /users/update/:id Updates corresponding user to id specified
DELETE /users/delete/:id Deletes corresponding user to id specified
HTTP Method Path Description
GET /products Fetches all Products
GET /products/:id Fetches corresponding product to id specified
POST /products/addProduct Adds a new product
PATCH /products/update/:id Updates corresponding product to id specified
DELETE /products/delete/:id Deletes corresponding product to id specified
